## Overview

Encrypt and decrypt text using Caesar's cipher. Supports multiple languages.

_This program was created for a homework assignment at LaunchCode's Lc101 (2018)._

## Technologies Used

-   Python
-   TypeScript
-   JavaScript
-   Flask
-   React
-   HTML
-   CSS
-   Vite
-   Axios
-   Tanstack React Query
-   pyspellchecker
